Practical-25 : package without parameter

p-25 folder > abc folder > abc.java




package abc;
public class abc
{
    public void display()
    {
        System.out.println("Hello");
    }
}

p-25 folder > demo.java



import abc.abc;

class demo
{
    public static void main(String args[])
    {
        abc aa = new abc();
        aa.display();
    }
}

Output:

Hello
