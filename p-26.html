Practical-26 : package with parameter

p-26 folder > abc folder > abc.java




package abc;
public class abc
{
    public void display(int a,int b)
    {
        System.out.println("Ans="+(a+b));
    }
}

p-26 folder > demo.java



import abc.abc;
class demo
{
    public static void main(String args[])
    {
        abc aa=new abc();
        aa.display(2,3);
    }
}

Output:

Ans=5
