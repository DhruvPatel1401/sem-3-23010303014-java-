Practical-31 : Thread 


class abc extends Thread
{
    public void run()
    {
        for (int i = 0; i <= 10; i++)
        {
            System.out.println(i);
        }
    }
}

class xyz extends Thread
{
    public void run()
    {
        for (int j = 11; j <= 20; j++)
        {
            System.out.println(j);
        }
    }
}

class demo
{
    public static void main(String args[])
    {
        abc a = new abc();
        xyz b = new xyz();
        a.start();
        b.start();
    }
}

Output:

The output will display numbers from 0-10 (thread abc) and 11-20 (thread xyz).
Example:
0
1
11
2
3
12
...
Output order may vary due to thread scheduling.
